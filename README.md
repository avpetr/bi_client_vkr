# BI Dashboard Constructor - Прототип BI системы

Интерактивный конструктор дашбордов для бизнес-аналитики с поддержкой множества типов графиков и источников данных.

## 🚀 Возможности

### ✨ Основные функции

- **Интерактивный Drag-and-Drop** - Перетаскивайте и изменяйте размер виджетов на сетке
- **6 типов графиков**:
  - 📈 Линейный график
  - 📊 Столбчатая диаграмма
  - 📉 Площадной график
  - 🥧 Круговая диаграмма
  - 🔵 Точечная диаграмма
  - 📊 Комбинированный график
- **8 наборов данных** для экспериментов:
  - Продажи по месяцам
  - Аналитика пользователей
  - Категории товаров
  - Источники трафика
  - Региональные продажи
  - Финансовые показатели
  - Удовлетворенность клиентов
  - Эффективность отделов
- **Множественные метрики** - Отображайте несколько показателей на одном графике
- **Автосохранение** - Ваш дашборд сохраняется в localStorage
- **Примеры дашбордов** - Быстрый старт с готовым примером

## 🛠️ Технологический стек

- **React 19** - Современная библиотека для создания UI
- **Material-UI (MUI)** - Компоненты интерфейса
- **Recharts** - Библиотека графиков для React
- **react-grid-layout** - Drag-and-Drop сетка с изменяемым размером
- **Vite** - Быстрая сборка и разработка
- **Axios** - HTTP клиент (готов для подключения бэкенда)

## 🎯 Использование

### Создание дашборда

1. **Нажмите "Добавить график"** или "Пример" для быстрого старта
2. **Выберите источник данных** из 8 доступных наборов
3. **Выберите тип визуализации** (линейный, столбчатый, круговой и т.д.)
4. **Выберите метрики** для отображения (можно несколько)
5. **Нажмите "Добавить график"**

### Управление виджетами

- **Перетаскивание**: Захватите виджет за заголовок и перетащите
- **Изменение размера**: Потяните за правый нижний угол виджета
- **Удаление**: Нажмите крестик в правом верхнем углу виджета

### Сохранение и загрузка

- **Автосохранение**: Дашборд автоматически загружается при открытии
- **Кнопка "Сохранить"**: Явное сохранение текущей конфигурации
- **Кнопка "Пример"**: Загрузка демонстрационного дашборда

## 📊 Наборы данных

### 1. Продажи по месяцам (`salesPerformance`)
Ежемесячная статистика продаж, прибыли, расходов и выручки
- `sales` - Объем продаж
- `profit` - Прибыль
- `expenses` - Расходы
- `revenue` - Выручка

### 2. Аналитика пользователей (`userAnalytics`)
Статистика активности пользователей
- `activeUsers` - Активные пользователи
- `newUsers` - Новые регистрации
- `churnedUsers` - Ушедшие пользователи
- `sessions` - Количество сессий

### 3. Категории товаров (`productCategories`)
Производительность по категориям товаров
- `value` - Общая стоимость
- `orders` - Количество заказов
- `avgPrice` - Средняя цена

### 4. Источники трафика (`trafficSources`)
Распределение посетителей по источникам
- `value` - Процент от общего трафика
- `visitors` - Количество посетителей
- `bounceRate` - Показатель отказов

### 5. Региональные продажи (`regionalSales`)
Продажи по регионам России
- `sales` - Объем продаж
- `customers` - Количество клиентов
- `avgOrder` - Средний чек

### 6. Финансовые показатели (`financialMetrics`)
Квартальная финансовая отчетность
- `revenue` - Выручка
- `costs` - Затраты
- `profit` - Прибыль
- `margin` - Маржинальность (%)

### 7. Удовлетворенность клиентов (`customerSatisfaction`)
Метрики обратной связи от клиентов
- `satisfaction` - Оценка удовлетворенности (1-10)
- `nps` - Net Promoter Score
- `responses` - Количество ответов
- `complaints` - Жалобы

### 8. Эффективность отделов (`employeePerformance`)
Производительность и удовлетворенность по отделам
- `productivity` - Производительность (%)
- `satisfaction` - Удовлетворенность (1-10)
- `headcount` - Численность

## 🔌 Интеграция с бэкендом

Проект готов к подключению бэкенда. В файле `src/services/api.js` закомментированы примеры API запросов:

```javascript
// Сохранение конфигурации дашборда
export const saveDashboardConfig = async (config) => {
    const response = await apiClient.post('/dashboards', config);
    return response.data;
};

// Получение метрик из ClickHouse/PostgreSQL
export const fetchMetrics = async (datasetId, filters = {}) => {
    const response = await apiClient.get(`/metrics/${datasetId}`, { params: filters });
    return response.data;
};
```

### Планируемые API endpoints:

- `POST /api/dashboards` - Создание/сохранение дашборда
- `GET /api/dashboards/:id` - Загрузка конфигурации дашборда
- `GET /api/metrics/:datasetId` - Получение данных для графиков
- `GET /api/datasets` - Список доступных источников данных

## 🎨 Настройка

### Изменение цветовой схемы

В `src/components/SmartWidget.jsx`:

```javascript
const COLORS = ['#8884d8', '#82ca9d', '#ffc658', '#ff7c7c', ...];
```

### Добавление нового типа графика

1. Добавьте новый case в `SmartWidget.jsx` в функцию `renderChart()`
2. Добавьте опцию в `AddWidgetDialog.jsx` в Select для типа визуализации
3. Импортируйте необходимый компонент из recharts

### Добавление нового набора данных

В `src/services/mockDatasets.js`:

```javascript
export const myNewDataset = [
    { name: 'Значение 1', metric1: 100, metric2: 200 },
    // ...
];

// Добавьте в datasetRegistry
export const datasetRegistry = {
    // ...
    myNewDataset: {
        id: 'myNewDataset',

        data: myNewDataset,
        metrics: [
            { key: 'metric1', label: 'Метрика 1', color: '#8884d8' },
            { key: 'metric2', label: 'Метрика 2', color: '#82ca9d' }
        ],
        xAxisKey: 'name'
    }

```
